# agenda-steph

Ce dépôt contient une application web à page unique (SPA) qui peut être testée en local avec un simple serveur web. Si vous n'êtes pas familier avec la programmation, suivez ces étapes :

1. Ouvrez le terminal (une application de ligne de commande). Sur Windows recherchez "Invite de commandes" ou "PowerShell", sur macOS/Linux utilisez "Terminal".
2. Déplacez-vous dans le dossier du projet :

   ```bash
   cd path/to/agenda-steph
   ```

   Par exemple : `cd ~/workspace/agenda-steph`.
3. Lancez le serveur en tapant :

   ```bash
   ./start.sh
   ```

   Si vous voyez un message indiquant « Serving /workspaces/agenda-steph on http://localhost:8000 », le serveur tourne.
4. Ouvrez votre navigateur web (Chrome, Firefox, Safari, etc.) et rendez-vous à l'adresse :

   `http://localhost:8000`

   La page de l'application s'affiche alors.
5. Pour arrêter le serveur, revenez dans le terminal et appuyez sur <kbd>Ctrl+C</kbd>.


Ces instructions fonctionnent sur la plupart des systèmes (Windows, macOS, Linux) et ne requièrent aucune connaissance en programmation ; il suffit de copier/colle les commandes ci-dessus.
